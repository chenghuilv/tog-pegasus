cimuser

Name

cimuser - add, modify, remove or list users in CIM passwords file.

Synopsis

cimuser -a -u username [ -w password ]

cimuser -m -u username [ -w password ] [ -n newpassword ]

cimuser -r -u username

cimuser -l

Remarks

This command does not configure or list authorization information (see cimauth (1M)).

Description

The cimuser command provides a command line interface to manage CIM users stored in the CIM passwords file. The location of the password file can be viewed using the cimconfig (1M) command.

The first form of cimuser allows addition of a new user to the CIM passwords file. This form of the command can only be used to add one user at a time. If the password is not specified at the command line, the user will be prompted for it.

The second form allows modification of the specified user's password in the CIM passwords file. This form of the command can only be used to modify one user at a time. If the old or new password is not specified at the command line, the user will be prompted for it.

The third form allows removal of the specified user from the CIM passwords file. This form of the command can only be used to remove one user at a time.

The last form of this command allows user names in the CIM passwords file to be listed. Specifying no options with the cimuser command is equivalent to specifying the -l option, which provides a listing of user names.

Options

The cimuser command recognizes the following options:
-a                           Indicates that a user is to be added in the password file.

-m                           Indicates that a user's password  is to be modified in the password file.

-r                           Indicates that a user is to be removed from the password file.

-l                           Displays the names of the users in the password file.

-u username          Indicates a specific user name.

-w password          Indicates the password for the specified user name.

-n newpassword    Indicates the new password for the specified user name.

Return Values

cimuser returns one of the following values:
0         Successful completion.

1         Error

When an error occurs, an explanatory error message is written to stderr and an error value 1 is returned.

Examples

The following command adds user "guest", a new CIM user to the CIM passwords file. The password for this user is "guest".
cimuser -a -u guest -w guest

The following command adds user "guest", a new CIM user to the CIM passwords file. Since the password is not specified the user will be prompted for it.

cimuser -a -u guest
Please enter your password:
Please re-enter your password:

The following command modifies an entry in the password file for user "guest", with the new password "foo1".
cimuser -m -u guest -w foo -n foo1
The following command removes the CIM user "guest" from the CIM passwords file.
cimuser -r -u guest
The following command displays the list of CIM user names.
cimuser -l
