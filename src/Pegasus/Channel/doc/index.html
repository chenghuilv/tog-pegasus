Classes

  • ChannelAcceptor The ChannelAcceptor is a server-side object.
  • ChannelConnector The ChannelConnector is a client-side object used to create and connect a channel to a server (see the connect() method).
  • ChannelHandler The user of the channel module derives from the ChannelHandler class to provide implementations of the handleOpen(), handleInput(), and handleClose() methods.
  • ChannelHandlerFactory The ChannelHandlerFactory class must be derived from and implemented by the user.
  • DefaultChannelHandlerFactory The DefaultChannelHandlerFactory<> template class provides an easy way to provide a ChannelHandlerFactory implementation in cases in which the ChannelHandler has a simple default constructor (rather than one with arguments)
