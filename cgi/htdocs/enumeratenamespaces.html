enumerateNameSpaces

This request enumerates the namespaces defined in the repository. This is not a CIM intrinsic method.

NameSpace   Defined the root namespace where the __Namespace class is defined.

ClassName    Defined the __Namespace class within the repository

LocalOnly                         If true only CIM Elements overridden within definition of class are returned.

IncludeQualifiers              If checked, show Qualifiers for class and returned Properties, Methods, and Method Parameters.

IncludeClassOrigins         If checked (TRUE), CLASSORIGIN attribute will be returned for all appropriate elements

PropertyList NOTE: NOT IMPLEMENTED Null or array of property names to filter request. See below.

          

Today the namespace manipulation defined in CIM Operations over HTTP is proprietary in nature.  It is based on a class named __Namespace which creates an instance for each namespace defined for the CIMOM. The only property in the __namespace class is the name.

Within Pegasus, the namespace manipulation based on the __namespace class can be executed through this HTML page or through the enumerateInstanceNames operation with the classname = __namespace and the namespace parameter =  root.  While it has not been formally defined within the specification, Pegasus today maintains the __namespace class in the root namespace.

Operations on the __NameSpace class are handled within Pegasus by a special provider that manipulates the instances of the class and also creates and deletes namespaces.
